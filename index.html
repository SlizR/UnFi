Get UnFi — quick starter

Copy this starter and adapt to your target. This is a minimal example showing how UnFi hands off to a verified payload — real production code will require platform-specific glue and key provisioning. 


#include "unfi.h"
int main(void){
  // init minimal hw
  hw_init();
  // verify payload signature
  if(unfi_verify_image("payload.bin", key_store_get_active())){
    // quick integrity & boot
    unfi_boot("payload.bin");
  } else {
    // fallback to recovery or safe-mode
    recovery_mode();
  }
  while(1) { /* should never get here */ }
}
